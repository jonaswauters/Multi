# Barcode Matcher (Web)

Webversie van een Excel/VBA barcode-matcher.

## Features
- Gebruiker scan: enkel letters vóór eerste spatie (uppercase)
- Barcode 1 + 2: vergelijking op eerste 9 tekens
- Match of via substitutie-lijst
- Status UI: incomplete / Match / No match
- Archief met timestamp + export CSV
- Persistente opslag via localStorage

## Run
Open `index.html` in je browser (geen server nodig).
